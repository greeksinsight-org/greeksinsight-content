# Profit from Time Decay

Time decay (Theta) represents the rate at which an option loses value as expiration approaches.

We can exploit this by:
- Selling OTM options with high IV
- Using spreads to limit margin
- Tracking volatility squeezes for timing entries

> Combine deep learning predictions with these strategies for maximum alpha.
