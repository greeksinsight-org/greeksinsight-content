# Deep Learning and Options

Machine learning models like LSTMs and Transformers can predict volatility shifts, helping traders anticipate **time decay** and **implied volatility crush**.

Key techniques include:
- Using historical option chains as time-series
- Predicting volatility squeeze phases
- Estimating probability of expiring worthless (POEW)
